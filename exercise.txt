1. Explain what the terminal is in your own words.

    The terminal is an interface that uses commands to perform actions such as creating directories and files.
    Without the use of a graphic user interface, it allows you to keep your hands on the keys to make your coding faster by not relying on mouse clicks.

3.  What are the steps to fork a GitHub repository?
    A fork is pulling someones code into your own files for use.
    You'll need to create a fork, clone and pull to your code.
    After making your changes, you'll commit and push the code to your repository.
